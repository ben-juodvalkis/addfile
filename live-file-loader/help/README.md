# Help

This folder contains the help patch for live-file-loader.

## Required Files (Create in Max)

### live-file-loader.maxhelp

Help patch structure:
1. Title and version banner
2. Quick description
3. Basic usage example with [live-file-loader]
4. All commands demonstrated with examples
5. Outlet descriptions
6. Links to example patches
7. Troubleshooting section

**Tips:**
- Use `[tab]` to organize sections
- Include copy-paste ready message boxes
- Show typical connections to Live API
